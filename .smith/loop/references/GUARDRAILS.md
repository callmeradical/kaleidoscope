# Guardrails Reference (Signs)

A sign captures a recurring failure and the corrective instruction.

## Anatomy of a Sign

### Sign: [Descriptive Name]
- **Trigger**: When this situation occurs
- **Instruction**: What to do instead
- **Added after**: When/why this was added
- **Example**: Concrete example if helpful

## Usage
- Add signs when the same mistake happens twice.
- Refine signs when they are too vague.
- Retire signs when no longer relevant.
